# doopen
This is a project of intelligent home appliance!
这是一个连接和控制家电的项目，有共同需求的朋友们可以借鉴下，主要实现了如下功能：


1：配对连接设备上网，

2:用户机制（登陆，注册，修改密码）

3:设备控制（控制：净水器，空气净化器。。。）

4:设备分享其他用户，

5:设备实时状态更新，
